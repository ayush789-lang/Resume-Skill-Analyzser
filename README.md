# Resume Skill Analyzer

This project analyzes a resume and extracts important skills using Python.
It helps identify the skills present in the resume and suggests improvements.

## Features

* Extract skills from resume text
* Analyze important technical skills
* Suggest missing skills for improvement

## Technologies Used

* Python
* NLP (Natural Language Processing)
* Pandas

## Project Structure

resume-skill-analyzer/
│
├── main.py
├── skill_analyzer.py
├── requirements.txt
└── README.md

## How to Run the Project

1. Install the required libraries

pip install -r requirements.txt

2. Run the program

python main.py
